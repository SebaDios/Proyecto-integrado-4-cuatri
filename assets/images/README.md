# Carpeta de Imágenes

Esta carpeta contiene las imágenes del sistema, incluyendo el logo de la empresa.

## Logo de la Empresa

Para agregar el logo de Antojitos Alkase:

1. Coloca el archivo del logo en esta carpeta con el nombre: `logo.png`
2. El logo debe tener un tamaño recomendado de 60x60 píxeles o proporcional
3. Formatos soportados: PNG, JPG, SVG
4. Si el logo no se encuentra, se mostrará un placeholder circular con la letra "A"

El sistema buscará automáticamente el logo en: `assets/images/logo.png`
